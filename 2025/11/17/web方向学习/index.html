请求方式

  1. GET：获取资源（参数在URL）
  2. POST： 提交数据（参数在请求体中）
  3. PUT： 更新资源
  4. DELETE： 删除资源

状态码含义

  • 200：请求成功
  • 301/302：资源重定向
  • 404：请求的资源不存在
  • 500：服务器内部错误

应用可参考[Week1] from_http

基础知识(HTML&CSS)

HTML

HTML文档的后缀：.html .htm .shtml

大部分浏览器中直接输出中文会乱码乱码，需要在头部将字符声明为 UTF-8 或 GBK (VScode 中用c++输入中文后运行也会吗？还是只是单纯的没做好环境配置)

一些介绍

  • <!DOCTYPE html>：声明当前文档为 HTML5 文档
  • <html>：HTML 页面的根元素（所有内容需包裹在该标签内）
  • <head>：包含文档的元数据，例如 <meta charset="utf-8">（定义页面编码为 UTF-8，解决中文乱码）
  • <title>：定义文档的标题（显示在浏览器标签栏）
  • <body>：包含页面的可见内容
  • <h1>：定义一个大标题
  • <p>：定义一个段落

注：在网页空白处右键，选择“网页源文件”或按 F12，均可查看网页源码

基础示例

<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>HTML基础示例</title>
  </head>
  <body>
    <h1>我是标题1</h1>
    <h2>我是标题2</h2>
    <p>这是一个段落内容</p>
    
    <!-- 链接示例 -->
    <a href="">这是一个链接</a>
    
    <!-- 图像示例（需确保路径与博客资源目录一致） -->
    <img src="图片路径" width="258" alt="示例图像">
    
    <!-- 表格示例 -->
    <table border="1">
      <tr>
        <td>表格单元格内容</td>
      </tr>
    </table>
  </body>
</html>
